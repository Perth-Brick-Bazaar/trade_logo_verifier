## Getting Started

1. Clone the repo:
   ```
   git clone https://github.com/Perth-Brick-Bazaar/trade_logo_verifier.git
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the verifier:
   ```
   python main.py
   ```
